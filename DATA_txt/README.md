# 🚗 ProTires Bot - Инструкция пользователя

> Современный помощник для подбора идеальных шин в Telegram

## 🤖 Что такое ProTires Bot?

**ProTires Bot** — это умный помощник в Telegram, который поможет вам найти идеальные шины для вашего автомобиля. Бот проведет вас через простой процесс выбора, учтет все ваши предпочтения и предложит лучшие варианты.

## 🔍 Быстрый старт

### Шаг 1: Найдите бота
```
@ProTiresBot
```

### Шаг 2: Запустите бота
```
/start
```

### Шаг 3: Выберите тип поиска
- **По размеру шин** — если знаете точные размеры
- **По марке автомобиля** — если хотите подобрать по модели авто

---

## ⚙️ Процесс подбора шин (5 шагов)

> 📋 **Важно:** Процесс подбора состоит из 5 последовательных шагов. Каждый шаг важен для точного подбора шин.

### 🔸 Шаг 1: Основные параметры
- Размер шин (ширина/высота/диаметр)
- Или марка и модель автомобиля

> ⚠️ **Внимание:** Убедитесь, что указываете правильные данные — от этого зависит точность подбора!

### 🔸 Шаг 2: Тип шин
- Летние шины
- Зимние шины  
- Всесезонные шины

### 🔸 Шаг 3: Дополнительные параметры
- Производитель (бренд)
- Ценовая категория
- Особые требования

### 🔸 Шаг 4: Подтверждение выбора
- Просмотр сводки по выбранным параметрам
- Внесение корректировок при необходимости
- Подтверждение поиска

### 🔸 Шаг 5: Результаты поиска
- Список подходящих шин с характеристиками
- Цены и наличие на складе
- Возможность связаться с менеджером
- Дополнительная информация по каждой модели

> 🎯 **Результат:** Вы получите персонализированную подборку шин, идеально подходящих под ваши требования.
